# issueops

This repository is a dedicated test bed for custom Handlebars helpers in [nmoa/validator](https://github.com/nmoa/validator).

It includes:
- a sample IssueOps member-request form
- custom validators and Handlebars helpers under `.github/validator/`
- a workflow pinned to `nmoa/validator@feat/custom-handlebars-helpers` while that functionality is being validated
